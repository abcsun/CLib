Detailed Description

The structure used to configure a JSON parser object.

Parameters
depthIf negative, the parser can parse arbitrary levels of JSON, otherwise the depth is the limit
Pointerto a callback. This parameter may be NULL. In this case the input is merely checked for validity.
Callbackcontext. This parameter may be NULL.
depth.Specifies the levels of nested JSON to allow. Negative numbers yield unlimited nesting.
allowComments.To allow C style comments in JSON, set to non-zero.
handleFloatsManually.To decode floating point numbers manually set this parameter to non-zero.
Returns
The parser object.
