Detailed Description

Core of message interaction with NOX.

The class allows messages to be sent and receive on the datapath in NOX. TCP and SSL sockets can be used. Binary messages with messenger_msg as the header are used in this communication.

All messenger or communication component must not use type 0x00 to 0x09 as there are reserved for messenger_core's internal use. Further, echo response (type 0x02) has to be implemented for end-to-end check with the component.
